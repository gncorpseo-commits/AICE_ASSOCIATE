# 📓 Jupyter Notebook 사용법 완전 가이드 --- ## 1️⃣ 새 노트북 생성 ### 방법 1: 메인 화면에서 ``` 1. Jupyter 메인 화면 (파일 목록 화면) 2. 오른쪽 상단 [New ▼] 버튼 클릭 3. [Python 3 (ipykernel)] 선택 4. 새 노트북이 열림! ``` ### 방법 2: 특정 폴더에서 ``` 1. 원하는 폴더로 이동 (폴더 클릭) 2. [New ▼] → [Python 3] 3. 해당 폴더에 노트북 생성됨 ``` --- ## 2️⃣ 노트북 이름 변경 ``` 1. 상단의 "Untitled" 클릭 2. 새 이름 입력 (예: "my_practice") 3. [Rename] 클릭 ``` --- ## 3️⃣ 셀(Cell) 이해하기 노트북은 **셀** 단위로 구성됩니다: | 셀 타입 | 용도 | 표시 | |---------|------|------| | **Code** | 파이썬 코드 실행 | `In [ ]:` | | **Markdown** | 설명, 제목 작성 | 텍스트 | ### 셀 타입 변경 - 상단 드롭다운에서 `Code` ↔ `Markdown` 선택 - 또는 단축키: `Y` (Code), `M` (Markdown) --- ## 4️⃣ 핵심 단축키 ⭐ ### 실행 관련 | 단축키 | 기능 | |--------|------| | **`Shift + Enter`** | 셀 실행 & 다음 셀로 | | **`Ctrl + Enter`** | 셀 실행 (제자리) | | **`Alt + Enter`** | 셀 실행 & 아래에 새 셀 | ### 모드 전환 | 단축키 | 기능 | |--------|------| | **`Enter`** | 편집 모드 (셀 안으로) | | **`Esc`** | 명령 모드 (셀 밖으로) | ### 셀 추가/삭제 (명령 모드에서) | 단축키 | 기능 | |--------|------| | **`A`** | 위에 새 셀 추가 | | **`B`** | 아래에 새 셀 추가 | | **`DD`** | 셀 삭제 (D 두 번) | | **`Z`** | 삭제 취소 | ### 셀 편집 | 단축키 | 기능 | |--------|------| | **`C`** | 셀 복사 | | **`X`** | 셀 잘라내기 | | **`V`** | 셀 붙여넣기 | ### 기타 | 단축키 | 기능 | |--------|------| | **`Ctrl + S`** | 저장 | | **`Ctrl + Z`** | 되돌리기 (편집모드) | | **`H`** | 단축키 도움말 (명령모드) | --- ## 5️⃣ 실습: 첫 노트북 만들기 ### Step 1: 새 노트북 생성 ``` [New ▼] → [Python 3] ``` ### Step 2: 이름 변경 ``` "Untitled" 클릭 → "practice_01" 입력 → Rename ``` ### Step 3: 첫 번째 셀에 코드 입력 ```python print("Hello, AICE!") ``` ### Step 4: 실행 ``` Shift + Enter ``` ### Step 5: 결과 확인 ``` Hello, AICE! ``` --- ## 6️⃣ 연습용 노트북 활용법 ### 🎯 추천 연습 방식 ``` 1. [New] → Python 3 로 새 노트북 생성 2. 이름: "typing_practice_01" 3. 아래 코드를 안 보고 타이핑 4. Shift+Enter로 실행하며 확인 ``` ### 연습할 코드 (순서대로) **셀 1:** ```python import numpy as np import pandas as pd ``` **셀 2:** (A 또는 B로 새 셀 추가) ```python df = pd.read_csv('../mock_exams/data/mock_01_delivery.csv') df.head() ``` **셀 3:** ```python df.shape ``` **셀 4:** ```python df.info() ``` **셀 5:** ```python df.isna().sum() ``` --- ## 7️⃣ 저장 & 종료 ### 저장 - **`Ctrl + S`** 또는 💾 아이콘 클릭 - 자동 저장도 됨 (체크포인트) ### 노트북 닫기 ``` [File] → [Close and Halt] ``` > ⚠️ 그냥 브라우저 탭 닫으면 커널이 계속 실행됨 ### Jupyter 완전 종료 ``` 터미널에서 Ctrl + C 두 번 ``` --- ## 8️⃣ 자주 쓰는 기능 ### 모든 셀 실행 ``` [Cell] → [Run All] ``` ### 출력 지우기 ``` [Cell] → [All Output] → [Clear] ``` ### 커널 재시작 (문제 발생 시) ``` [Kernel] → [Restart] ``` ### 커널 재시작 & 모든 셀 실행 ``` [Kernel] → [Restart & Run All] ``` --- ## 9️⃣ 연습 팁 ### 효율적인 연습 루틴 ``` 📁 basics 폴더에서: 1. pandas_basics.ipynb 열기 (참고용) 2.
